# TCG Favorites Dashboard

Static GitHub Pages snapshot of the Sneaker Dunk TCG dashboard.

Open `index.html` through GitHub Pages.

Notes:

- Data is loaded from `tcg_data_cumulative.js`.
- This static page does not run the Sneaker Dunk source updater.
- To refresh online data, run the local dashboard updater first, then publish the updated `docs/` files.
